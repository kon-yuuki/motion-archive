Technical memo

Hover Intentは、ポインターが一瞬通過しただけの状態と、ユーザーが意図して対象に留まった状態を区別する考え方です。 初回だけ表示を遅らせ、一度操作が始まったグループ内では即時に切り替えます。 

const delay = groupActive ? 0 : openDelay;

if (delay === 0) {
  open(target);
} else {
  openTimer = setTimeout(() => open(target), delay);
}

カーソルの右側にツールチップを置く前に、表示領域の端までの空きを測ります。 必要な幅を確保できない場合だけ左側へ反転し、内容が見切れない位置へ配置します。 

const spaceRight = boundary.right - pointerX;
const requiredWidth = tooltip.offsetWidth + gap;

tooltip.dataset.side =
  spaceRight >= requiredWidth ? "right" : "left";
