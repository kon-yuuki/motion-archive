Implementation memo

Shape

ボタン幅はラベルとアイコンの実寸に合わせる。右上と左下だけを欠いたアウトラインにし、背景が透過のバリエーションではホバー後も塗りつぶさない。

Hover sequence

ホバーすると、まず短い線分が上中央と下中央から同時に走る。どちらも時計回りに進み、最大30px程度まで伸びたあと、互いの開始位置付近へ移動しながら縮んで消える。

同時にラベルは1文字目だけを固定し、2文字目以降を一度消してから左から順にシャッフル表示する。ホバー解除が途中で入っても、開始済みのアニメーションは最後まで再生する。

Border easing

走る線は、位置移動と線の長さを別々のアニメーションに分ける。位置移動はfrom/toだけにして中間停止を避け、線の長さだけを0%/50%/100%で伸縮させる。

イージングはexpo系のease-in-outに寄せる。ジワッと動き出して中央付近で一気に進み、終わり際はゆっくり縮んで消えるが、最大長のまま移動する区間は作らない。

SVGの線端を常にroundにすると、線の長さが0になった瞬間にも丸い端点だけが小さく残って見える。線が出ている間だけroundにし、開始前と終了後はbuttに切り替えて、不要な点が描画されないようにする。

Mark motion

左の4ブロックは2段階だけ動かす。1回目で元とは違う縦長の中間形状になり、2回目で同じ方向へ進んで結果的に元の形状へ戻る。

Layout stability

シャッフル中に文字幅が変わるとボタン全体が揺れるため、不可視の固定ラベルで幅を確保し、実際に動くラベルはその上にabsoluteで重ねる。
