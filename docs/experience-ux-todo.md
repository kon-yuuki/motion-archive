# Experience UX Audit & TODO

監査日: 2026-06-13

## 目的

このTODOはアクセシビリティやHTML構造ではなく、以下の体験品質を改善するためのものです。

- 初見理解
- 視覚的な優先順位
- 操作の発見しやすさ
- 作品の探しやすさと比較しやすさ
- 詳細ページ閲覧後の回遊
- モーションフィードバックの品質
- モバイルでの情報密度

## 監査範囲

- サイトトップ
- Motion Archiveトップ
- Categories
- UI Galleryトップ
- Tooltip Behavior
- Typography
- Easings
- 作品詳細ページ
- 390px / 844pxのモバイル表示
- 1440px / 900pxのデスクトップ表示

## 主な監査結果

1. ヒーローの存在感が強く、主要コンテンツへの到達が遅い。
2. Motion ArchiveとAll Worksの内容がほぼ同じで、階層を分ける価値が弱かった。
3. 作品一覧が文字とタグ中心で、作品の見た目や動きを比較できない。
4. 詳細ページで操作方法とInfoボタンを自力で発見する必要がある。
5. 詳細ページを見終えた後の次作品、一覧、関連作品への導線がない。
6. Easing Indexは30件を一度に並べ、モバイルのページ高が約14,155pxになる。
7. UI Galleryでは公開済み2件と未公開2件が同じ強さで表示される。
8. テキストリンク、カード、主要ボタンに同じ押下縮小が適用され、操作の重要度が伝わりにくい。
