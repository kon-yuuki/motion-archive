技術メモ

CSS Pie Chart

親要素から番号を割り当てる

割合を data-percentage-1 のように親へ集約し、型付き attr() で数値としてCSSへ渡す。

割合を累積する

--accum-3 を「前の累積値 + 前の割合」として連鎖させ、各スライスの開始角度を求める。

円グラフを描画する

  • 各 li が透明部分を含む一つの conic-gradient() を描く。
  • 全スライスを同じグリッドセルに重ねて一つの円に見せる。
  • sin() と cos() でラベルを扇形の中央へ配置する。
  • ラジオボタンの状態だけで円グラフと棒グラフを切り替える。
  • ホバー位置の角度から対象スライスを判定し、詳細ツールチップを表示する。

編集GUI

GUI部分だけはJavaScriptを使用し、入力内容から親の属性と子のラベルを更新する。円グラフの角度計算と描画自体は引き続きCSSが担当する。

ブラウザ互換性

型付き attr() を使うため、未対応ブラウザではグラフが成立しない。実験ページとしてモダンCSSを優先している。
