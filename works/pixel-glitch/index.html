技術メモ

Pixel Glitch

表示領域への進入を検知する

各画像を IntersectionObserver で監視し、viewport の上下 10% を除いた領域へ入ったタイミングで一度だけ発火させる。

キャンバスで画像を段階的に描画する

画像の上に canvas を重ね、低解像度の scratch canvas に描いた画像を拡大して、大きいピクセルから細かいピクセルへ段階的に変化させる。

グリッチの質感を作る

  • 前半だけ横方向のスライスずれを入れる。
  • 終盤のフラッシュは止めて、静かに通常画像へ戻す。
  • 完了時に canvas を消して、本物の画像を表示する。
